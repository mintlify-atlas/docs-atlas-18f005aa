Baileys provides comprehensive TypeScript types for working with WhatsApp messages.

## Core Message Types

### WAMessage

Extends `proto.IWebMessageInfo` with additional Baileys-specific fields.

<ResponseField name="key" type="WAMessageKey" required>
  Unique identifier for the message
</ResponseField>

<ResponseField name="message" type="WAMessageContent">
  The actual message content (proto.IMessage)
</ResponseField>

<ResponseField name="messageTimestamp" type="number | Long">
  Unix timestamp when message was sent
</ResponseField>

<ResponseField name="messageStubParameters" type="any">
  Parameters for stub messages (system messages)
</ResponseField>

<ResponseField name="category" type="string">
  Message category classification
</ResponseField>

<ResponseField name="retryCount" type="number">
  Number of times message delivery has been retried
</ResponseField>

### WAMessageKey

Extends `proto.IMessageKey` with additional routing information.

<ResponseField name="remoteJid" type="string">
  JID of the chat (group or individual)
</ResponseField>

<ResponseField name="fromMe" type="boolean">
  Whether the message was sent by you
</ResponseField>

<ResponseField name="id" type="string">
  Unique message ID
</ResponseField>

<ResponseField name="participant" type="string">
  Sender's JID in group chats
</ResponseField>

<ResponseField name="remoteJidAlt" type="string">
  Alternative JID format (Baileys extension)
</ResponseField>

<ResponseField name="participantAlt" type="string">
  Alternative participant format (Baileys extension)
</ResponseField>

<ResponseField name="server_id" type="string">
  Server-assigned ID (Baileys extension)
</ResponseField>

<ResponseField name="addressingMode" type="string">
  Addressing mode: 'pn' or 'lid' (Baileys extension)
</ResponseField>

<ResponseField name="isViewOnce" type="boolean">
  Whether this is a view-once message (Baileys extension)
</ResponseField>

## Message Content Types

### AnyMessageContent

Union type for all possible message content:

```typescript
type AnyMessageContent =
  | AnyRegularMessageContent
  | { forward: WAMessage; force?: boolean }
  | { delete: WAMessageKey }
  | { disappearingMessagesInChat: boolean | number }
  | { limitSharing: boolean }
```

## Message Updates and Events

### MessageUpsertType

```typescript
type MessageUpsertType = 'append' | 'notify'
```

- **notify**: New message that should trigger a notification
- **append**: Historical message being synced, no notification needed

### WAMessageUpdate

<ResponseField name="key" type="WAMessageKey" required>
  Key identifying the message to update
</ResponseField>

<ResponseField name="update" type="Partial<WAMessage>" required>
  Partial message data with updates to apply
</ResponseField>

### MessageUserReceipt

Alias for `proto.IUserReceipt` containing:

<ResponseField name="userJid" type="string">
  User who sent the receipt
</ResponseField>

<ResponseField name="receiptTimestamp" type="number | Long">
  When the receipt was sent
</ResponseField>

<ResponseField name="readTimestamp" type="number | Long">
  When the message was read
</ResponseField>

<ResponseField name="playedTimestamp" type="number | Long">
  When media was played
</ResponseField>

</ResponseField>

<ResponseField name="directPath" type="string | null">
  Direct download path
</ResponseField>

<ResponseField name="url" type="string | null">

</ResponseField>

### MessageReceiptType

```typescript
type MessageReceiptType =
  | 'read'
  | 'read-self'
  | 'hist_sync'
  | 'peer_msg'
  | 'sender'
  | 'inactive'
  | 'played'
  | undefined
```

### MinimalMessage

Minimal message data for references:

```typescript
type MinimalMessage = Pick<WAMessage, 'key' | 'messageTimestamp'>
```

## Example Usage

```typescript
import { AnyMessageContent } from '@whiskeysockets/baileys'

// Send a text message
const textMsg: AnyMessageContent = {
  text: 'Hello @1234567890',
  mentions: ['1234567890@s.whatsapp.net']
}

// Send an image
const imageMsg: AnyMessageContent = {
  image: { url: 'https://example.com/image.jpg' },
  caption: 'Check this out!',
  mentions: ['1234567890@s.whatsapp.net']
}

// Send a poll
const pollMsg: AnyMessageContent = {
  poll: {
    name: 'What\'s your favorite color?',
    values: ['Red', 'Blue', 'Green'],
    selectableCount: 1
  }
}

// Delete a message
const deleteMsg: AnyMessageContent = {
  delete: messageKey
}

// Forward a message
const forwardMsg: AnyMessageContent = {
  forward: originalMessage,
  force: true
}
```
